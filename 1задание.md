2.генерация массива
```python
    def generate_array(n):
    arr = []
    for i in range(n):
        arr.append(random.randint(0, 10000))
    return arr
```
3.измерение времени
```python
    def measure_time(func, data,a):
    start = time.perf_counter()
    func(data,a)
    end = time.perf_counter()
    return end - start
```
