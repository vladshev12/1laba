2.измерение времени
```python
    def measure_time(func, data):
    start = time.perf_counter()
    func(data)
    end = time.perf_counter()
    return end - start
```
3.создание массива
```python
    def generate_array(n):
    arr = []
    for i in range(n):
        arr.append(random.randint(0, 10000))
    return arr
```
4.измерение памяти
```python
    def memory(func,data):
    tracemalloc.start()
    func(data)
    cur,peak=tracemalloc.get_traced_memory()
    tracemalloc.stop()
    return peak/1024
```
